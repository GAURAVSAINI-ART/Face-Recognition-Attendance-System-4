# Face Recognition Attendance System

## Requirements
- Python 3.8+
- Webcam

## How to Run

1. Open folder in VS Code
2. Open terminal
3. Create virtual environment:
   python -m venv venv

4. Activate it:
   venv\Scripts\activate

5. Install libraries:
   pip install -r requirements.txt

6. Run program:
   python main.py
